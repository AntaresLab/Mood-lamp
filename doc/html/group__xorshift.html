u16 get_random_u16 ( )

Random value getter.

Calculates new random value by 16-bit xorshift method:

u16 get_random_u16(){
static u16 y16 = 1;
y16 ^= (y16 << 13);
y16 ^= (y16 >> 9);
return y16 ^= (y16 << 7);
}
